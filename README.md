# Express Server Example

This example shows A simple Express project with mongoose 

## Getting started

### 1. Install dependencies

Download this example:

```
npm install
```

### 2. Start the Express server

Launch your Express server with this command:

```
npm run dev
```

or 

```
npm run start
```

### 3. Routes
* reservations/create
* reservations/pay
* reservations/list
* reservations/cancel

# Express Server Example

This example shows A simple express project mongoose 

## Getting started

### 1. Install dependencies

Download this example:

```
npm install
```

### 2. Start the express server

Launch your GraphQL server with this command:

```
npm run dev
```

or 

```
npm run start
```

### 3. Routes
* reservations/create
* reservations/pay
* reservations/list
* reservations/cancel
